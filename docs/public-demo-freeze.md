# Public Demo Freeze

## Status

Mergen Watch Base is frozen as a public proof-of-build demo for Mergen Watch League.

The public repo is intended to show product direction, frontend execution, Base ecosystem positioning, and the browser-game-style squad loop.

## Finished For Public Demo

- Premium dark Mergen Watch League interface
- Five-slot squad builder
- Token add/remove interaction
- Squad persistence with localStorage
- Mock wallet connect/disconnect flow
- Wallet-keyed mock watcher profile
- Avatar selection using existing token character assets
- Weekly XP, points, rank, and leaderboard panels
- English and Turkish UI support
- Public Vercel deployment
- Open GitHub demo repository

## Do Not Add To Public Repo

- Private keys, seed phrases, or production credentials
- Real wallet integration libraries
- Smart contracts or deploy scripts
- Main Mergen Finance production code
- Private product logic
- Real token indexing credentials
- Swap execution logic
- Production moderation systems
